# Deskripsi Aplikasi

Aplikasi ini adalah platform pembelajaran bahasa Inggris berbasis web yang dibangun menggunakan React (Vite), TypeScript, dan Firebase. Fitur utama meliputi:
- Autentikasi pengguna (login, register)
- Dashboard pengguna
- Pengelolaan materi pembelajaran
- Pengaturan profil
- Fitur admin untuk manajemen database, notifikasi, referral, dan reward

Struktur folder utama:
- `components/` — Komponen UI reusable
- `pages/` — Halaman utama aplikasi (user & admin)
- `firebase.ts` — Konfigurasi Firebase
- `types.ts` — Tipe data TypeScript

Aplikasi ini cocok untuk institusi atau individu yang ingin belajar dan mengelola materi bahasa Inggris secara online.
