# Daftar File di Root Project dan Penjelasannya

1. **App.tsx** — Entry point utama aplikasi React.
2. **firebase.json** — Konfigurasi Firebase Hosting dan deploy.
3. **firebase.ts** — Inisialisasi dan konfigurasi Firebase di aplikasi.
4. **firestore.rules** — Aturan keamanan database Firestore.
5. **index.html** — Template HTML utama (digunakan oleh Vite/React).
6. **index.tsx** — Entry point React, mounting ke DOM.
7. **metadata.json** — Metadata aplikasi (opsional, bisa untuk info build, dsb).
8. **package.json** — Daftar dependencies dan script npm.
9. **README.md** — Dokumentasi utama proyek.
10. **tsconfig.json** — Konfigurasi TypeScript.
11. **types.ts** — Definisi tipe data global TypeScript.
12. **vite.config.ts** — Konfigurasi build dan dev server Vite.
13. **components/** — Folder komponen UI reusable.
14. **docs/** — Dokumentasi tambahan (setup, deployment, dsb).
15. **pages/** — Halaman utama aplikasi (user & admin).
16. **public/** — (Belum ada, sebaiknya dibuat) Untuk file statis seperti gambar, HTML, dsb.
17. **tests/** — (Belum ada, opsional) Untuk unit test.
18. **.env.example** — (Belum ada, sebaiknya dibuat) Contoh environment variable.
19. **.gitignore** — (Belum ada, sebaiknya dibuat) Untuk mengabaikan file tertentu di git.
20. **LICENSE** — (Belum ada, opsional) Lisensi proyek.

File/folder yang belum ada bisa ditambahkan sesuai kebutuhan proyek.
