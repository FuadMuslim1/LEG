# File yang Kurang dari Aplikasi

1. **.env.example** — Contoh file environment untuk variabel rahasia (API key, dsb)
2. **CONTRIBUTING.md** — Panduan kontribusi untuk developer lain
3. **CODE_OF_CONDUCT.md** — Aturan perilaku komunitas
4. **LICENSE** — Lisensi proyek (MIT, GPL, dsb)
5. **tests/** — Folder untuk unit test/komponen
6. **public/** — Folder untuk aset statis (logo, gambar, dsb)
7. **.gitignore** — Untuk mengabaikan file yang tidak perlu di repo

Tambahkan file di atas sesuai kebutuhan proyek Anda.
